Computer Network Project

Source Code
May 2019 - Jun 2019

The main goal of the project is to recreate a computer network and assign two resources to every node in the network. The network contains exactly 30 hosts and 10 assignable resources. The assignment is managed in two distinct phases:

  • Phase 1: every host requests the desired resource needed to complete its task. A resource can receive more than one request, and they will store those requests in a dedicated list. The resource will be assigned to the host with the biggest priority. The rejected hosts will be discarded and they will need to make another request in the future to get that specific resource. This phase terminates when all the hosts have received at least one resource.
  • Phase 2: in this phase, every host will receive the second resource necessary to complete its tasks. Randomly the Allocator selects both a node in the network and a resource. Therefore the resource is assigned to the node. This phase terminates when every host has received the second resource.
