MIPS 32: Computer Architectures Final Project

Source Code
May 2017 - Jun 2017

The project is part of the class "Computer Architectures" of the coursework in Computer Science at the University of Florence. The project required the completion of three individual programs; the programming language used is the MIPS 32 Assembly. Should you see the main project page on my github account (Github) for a complete overview.

First Program: String Analysis

This program takes in input a string with a maximum length of 100 characters, and "traslates" each subsequence of characters (which represents a digit in italian language) separated by a blank space, with the respective numerical digit. Any subsequence of characters that doesn't represent a digit, will be translated with "?". Below we can see some examples respectively of inputs and outputs of the program:

  • uno due tre 4 -> 1 2 3 ?
  • ciao uno ciao tre cinque -> ? 1 ? 3 5

Third Program: Matrix Operations

This program allow the user to insert several values (maximum 4) to create two nxn matrices, required to perform most of the common used operations between them, respectively: the sum, the subtraction, the product. Thanks to a basic user interface, the user can choose the desired operation. In case we want to change the values to create two new matrices, we reuse those memory locations, overriding the previous values: this is made possible by storing in the stack the adressess of the matrices.
